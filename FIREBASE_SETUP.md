read by anyone, write by admin only\n    match /products/{document} {\n      allow read: if true;\n      allow write: if request.auth != null && \n        request.auth.token.admin == true;\n    }\n    \n    // User profiles - users can read/write their own data\n    match /users/{userId} {\n      allow read, write: if request.auth != null && \n        request.auth.uid == userId;\n    }\n    \n    // Orders - users can read their own orders\n    match /orders/{orderId} {\n      allow read: if request.auth != null && \n        (request.auth.uid == resource.data.userId || \n         request.auth.token.admin == true);\n      allow create: if request.auth != null && \n        request.auth.uid == request.resource.data.userId;\n    }\n  }\n}\n`\n\n### Storage Rules\n`javascript\nrules_version = '2';\nservice firebase.storage {\n  match /b/{bucket}/o {\n    // Product images - users can manage their own\n    match /users/{userId}/{allPaths=**} {\n      allow read: if true;\n      allow write: if request.auth != null && \n        request.auth.uid == userId;\n    }\n  }\n}\n`\n\n## Testing\n\nOnce configured, you should see:\n- âœ… \"Firebase initialized successfully\" in console\n- âœ… Products loading from Firestore\n- âœ… Admin can upload products\n- âœ… No more \"mock service\" messages\n\n## Troubleshooting\n\n### Common Issues\n\n1. **\"Firebase initialization failed\"**\n -
